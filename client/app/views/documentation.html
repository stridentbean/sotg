Search for Tweets
Find tweets by your specified keyword

Use our endpoint 'api/search' when you want to search our database for Tweets based on keyword. You can only specify one keyword at a time.

To get Tweets from our database:

GET
api/search

Parameters:
apiKey
Your API Key.
keyword
Keyword to run search.

Add and Delete Keywords
Keep track of and modify the keywords you request

Use our endpoint 'api/keywords' when you want our servers to track a keyword or if you want us to stop tracking a keyword you have already requested.

To add keywords:

POST
api/keywords

Parameters:
apiKey
Your API Key.
keyword
Keyword to run search.

DELETE
api/keywords

Parameters:
apiKey
Your API Key.
keyword
Keyword to run search.

Find Tweets by Sentiment
Discover whether keywords have positive or negative connotations on Twitter

Use our endpoint 'api/sentiment' to search for tweets containing a keyword based on whether or not they have a positive or negative connotation.

To find tweets containing keywords based on sentiment:

GET
api/sentiment

Parameters:
apiKey
Your API Key.
keyword
Keyword to run search.
sentiment
Can be positive or negative

Find Tweets by Time Range
See how keywords propagate over time

Use our endpoint 'api/time' to search for tweets containing a keyword over a period of time.

To find tweets containing keywords over a time range:

GET
api/time

Parameters:
apiKey
Your API Key.
keyword
Keyword to run search.
timeStart
The starting time, formatted as milliseconds since the UNIX epoch.
timeEnd
The ending time, formatted as milliseconds since the UNIX epoch.
